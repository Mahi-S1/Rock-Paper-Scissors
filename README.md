# Rock-Paper-Scissors
A fun and simple Rock, Paper, Scissors game designed to showcase DOM manipulation and CSS layout.

## Features 
Interactive UI : Smooth hover effects using CSS.
Smart Score Tracking : Keeps track of the player's and computer's score during the session.
Clean Code : Built using semantic HTML, organized CSS variables, and vanilla javascript.

## How It Works
